About me

A quick intro

I'm Juliet, a frontend developer, budding back-end developer and technical writer. I am a final year accountancy undergraduate.

I began my tech journey by learning Python to transition into data analysis but had to stop for some reasons. Early 2022, I started my web dev journey with front-end development with the aim of becoming a full-stack developer.

During the course of my front-end development journey, I have been able to acquire relevant technical and soft skills which grants me the ability to build efficient and aesthetically pleasing user interfaces, creating responsive and functional websites and web apps with modern web technologies, in line with modern best practices and at the same time working daily to acquire more skills and further improve myself.

Projects

Here are some projects I've worked on:

Emerald - React CRUD App

By creating events, reading events, updating events, and deleting events that have been entered at a previous date, users of this web app are able to keep track of their events. Built using React JS, Firestore, Tailwind CSS, React-Toastify & Framer motion.

Dockett

Dockett is a note-taking web application designed for just anyone with busy schedules. This is a collab project with a fellow front-end developer. It was developed using the following technologies React.js, React Slick, React-toastify, React-icons, Firebase, Tailwind, Material UI, and Framer Motion.

Rez Bloggz

A blogging web application that enables users to write and publish blog posts, and articles. It also comes with a delete function to enable users to delete the blog. Built using React JS, Framer motion, JSON and Tailwind CSS for styling.

Juls_Dash

Functional dashboard for a fictional organization that allows only the admin to login to the dashboard, input orders from customers and get notifications for orders inputted. It was created to improve my CSS and JavaScript skills.

EnDasmu NFT

This is an nft marketplace landing page for EnDasmu project. It was a team pair programming collab exercise project to practicalize the use of Git and Github. As the team lead, my role was assigning tasks, working on my code section, cordinating team members, debugging team mates codes during merging.

Oma Empire

A fictional shopping website dealing in female wears. Oma Empire provides amazing offers, add to cart section(though not functional yet) and enticing products. Built using HTML, CSS, JavaScript and animations

EatRite

A website for a fictional restaurant which provides online and offline services to customers, catering services, varities of meals to choose from, pay-on-delivery services and refunds on mistaken payments. Built using HTML, CSS and JavaScript.

Julie's Django ToDo App

Functional and responsive django todo web app. This is a full-stack project I built using html, css for the frontend UI and python and django for the backend functionalities. It enables users to add items and delete items.

Django Authentication System

A functional and responsive authentication website with Python and Django. This site restricts users' access to the full content of the page until their login has been verified. I also worked on the frontend interface using HTML and Tailwindcss to make it aesthetically pleasing to users..

Django Weather App

A Django weather application that retrieves local weather data when a city or country's name is entered. I was able to work with and implement Openweather API. Frontend interface was built using HTML and CSS and then the logic was built with Python and Django .

Recipe App

A React JS web application where users can input food name and get a listing of exact or similar food matches, together with some details about the food. This project was built to improve my useEffect hook and props skills and also to learn how to fetch data from a third-party API.

Prime Cuisine

This is a Desktop food ordering web application. Technologies used: React.js, Styled Components, Firebase, CSS, Framer motion, React-toastify & React icons. This project was built to practicalize the use of Reactjs concepts like: useState, useEffect, styled components and props
